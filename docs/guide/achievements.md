# Achievements

This game has an immense amount of content to explore and lore to uncover. As a result, the progress you make can feel a bit…intangible. To help combat this feeling, various achievements have been added to the game. These will be automatically unlocked as you explore and adventure.

!!! tip "Utilize the `[feats` command to access all of the achievements you or others have unlocked!"
