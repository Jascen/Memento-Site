# Game Clients

<https://github.com/PlayTazUO/TazUO/releases>

The recommended client is TazUO. Although the package you downloaded already has this ready to go, you can feel free to reach out to the developer for any updates or tech support.

Other clients, like ClassicUO, should function fine but there are two things you need to keep from this game's package:

- The "containers.txt" file in each of the client's Data\Client directory.

- The entire "Files" folder, in the main Game directory, that came with this package.

If you do not have either of these intact from this package, then your client will not function correctly.
