## Font too small (Windows)
If your text and gumps are too tiny to read, you can match the DPI scaling of the rest of your system.

- Go to `C:\Ultima-Memento\Client\TazUO-Launcher\TazUO`
- Right click `TazUO.exe`
!!! warning "Alternative Filename"
	The Memento package uses an old version of TazUO where the application was still called `ClassicUO.exe`
- Select *Properties*
- Select the *Compatability* tab
- Click the *Change high DPI settings* button
- In the *High DPI scaling override* section, check the box that says *Override high DPI scaling behavior. Scaling performed by:*
	- Make sure `System` is selected in the dropdown
